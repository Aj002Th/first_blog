# 简陋博客接口文档

## 1.用户

### 1.1注册

#### 请求说明

> 请求方式: POST
>
> 请求URL: /users

#### 入参

```
{
    "username": "hengheng",
    "password": "123456"
}
```

#### 参数

| 字段     | 字段类型 | 字段说明 |
| -------- | -------- | -------- |
| username | string   | 用户名   |
| password | int      | 密码     |

#### 返回结果

```
{
	"注册成功！"
}
```

#### 错误状态码

| 状态码（code） | 说明             |
| -------------- | ---------------- |
| 409            | 该用户名已被注册 |
